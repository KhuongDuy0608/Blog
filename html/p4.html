TẠO HIỆU ỨNG ĐỘNG VỚI JAVASCRIPT

JavaScript không chỉ là công cụ để xử lý logic phía máy chủ, mà còn giúp tạo ra những hiệu ứng trực quan và tương tác hấp dẫn trên trang web. Một trong những hiệu ứng đơn giản nhưng hiệu quả nhất là hiệu ứng hover, giúp thay đổi giao diện khi người dùng di chuột qua các phần tử. Bằng cách kết hợp JavaScript với CSS, bạn có thể tạo ra các hiệu ứng hover như thay đổi màu sắc, phóng to hình ảnh hoặc thêm các chuyển động mượt mà. Bên cạnh đó, việc cuộn trang lên đầu cũng là một tính năng rất hữu ích, đặc biệt đối với các trang dài. Bài viết này sẽ hướng dẫn bạn cách sử dụng JavaScript để tạo hiệu ứng hover và thêm tính năng cuộn lên đầu trang, giúp trang web của bạn trở nên sinh động và thân thiện với người dùng hơn.

Tạo hiệu ứng động với JavaScript

Bạn có thể sử dụng JavaScript để tạo ra các hiệu ứng thú vị trên website của mình, chẳng hạn như hiệu ứng hover động hoặc cuộn mượt. Dưới đây là một ví dụ:

Hiệu ứng hover

Thêm hiệu ứng hover động cho các liên kết hoặc nút bấm:

<a href="#" class="hover-effect">Hover me!</a>

Khi di chuột qua, bạn sẽ thấy hiệu ứng phóng to và đổi màu.

Ví dụ thực tế:

Hover me!

Thêm một số CSS để hỗ trợ hiệu ứng:


            <style>
                .hover-effect {
                    display: inline-block;
                    padding: 10px 20px;
                    background-color: lightblue;
                    color: black;
                    text-decoration: none;
                    border-radius: 5px;
                    transition: all 0.3s ease-in-out;
                }
                .hover-effect:hover {
                    background-color: blue;
                    color: white;
                    transform: scale(1.1);
                }
        
                #scroll-to-top {
                    position: fixed;
                    bottom: 20px;
                    right: 20px;
                    display: none;
                    padding: 10px;
                    background-color: black;
                    color: white;
                    border: none;
                    border-radius: 50%;
                    cursor: pointer;
                    font-size: 20px;
                }
            </style>

Cuộn mượt đến đầu trang

Thêm nút "Lên đầu trang" với hiệu ứng cuộn mượt:

<button id="scroll-to-top">↑</button>

Và đây là JavaScript để xử lý hiệu ứng cuộn:

// Hiển thị nút khi cuộn xuống
    window.addEventListener('scroll', () => {
        const scrollToTopButton = document.getElementById('scroll-to-top');
        if (window.scrollY > 200) {
            scrollToTopButton.style.display = 'block';
        } else {
            scrollToTopButton.style.display = 'none';
        }
    });
    
    // Cuộn mượt khi bấm nút
    document.getElementById('scroll-to-top').addEventListener('click', () => {
        window.scrollTo({
            top: 0,
            behavior: 'smooth'
        });
    });
